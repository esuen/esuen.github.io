What it does

Deep Dive is a content production pipeline that researches any topic, verifies every factual claim against sources, and converts the research into multiple output formats: a voiced podcast with per-line emotional delivery directions, a blog post with comparison tables and diagrams, and a video with frame-accurate synchronized visuals.

The research layer is format-agnostic—the same verified, confidence-tagged research produces a podcast, a blog post, or a presentation. Research once, output to multiple formats.

Under the hood

DAG-based orchestration with quality gates

16 Dagster software-defined assets with 8 quality checks at each stage. Whisper hallucination detection after voicing, adjacent-block contamination checks, content timing verification after video assembly, script quality audit for AI anti-patterns. Dagster tracks which outputs are stale and provides a web UI for pipeline observability.

Multi-platform output from one voicing pass

One full script (32 min) is voiced once ($2–3). An LLM cuts it to 22 min for YouTube by identifying the weakest sections (~$0.004)—audio spliced at block boundaries, no re-voicing. Spotify gets the full deep dive, YouTube gets a tighter edit optimized for algorithmic distribution. Blog posts and short clips derive from the same source.

Narration-synced animated visuals

24 Manim animated visuals per episode, synced to the exact word being spoken via Whisper word-level timestamps. Animations trigger on specific narration words—a map zooms out when the host says “geographically,” ship icons appear on “ships,” a stat card flashes on the key number. Script-line-aware matching prevents common-word false triggers. Drift compensation handles untracked camera animations. Templates: GridDashboard, PriceLine, ScaleComparison, MapScene (with matplotlib backgrounds), TypewriterQuote.

TTS hallucination prevention

Two-tier detection: duration check (free), then Whisper transcription verification (re-voice up to 3 times). A pre-voice transform converts period-separated lists to comma format—preventing TTS from hallucinating repeated words. Adjacent-block contamination check catches words leaking from one block into the next. Voice consistency profiling via Praat (F0, formants, HNR) detects and fixes gender/timbre switches.

Script quality enforcement

Automated quality check scans for AI writing anti-patterns: banned phrases, stacking emphasis (“not 2x, not 5x, 28x”), missing contractions, monotonous rhythm. The scriptwriter prompt encodes data presentation rules: every number needs comparison or analogy, one stat per beat, cold open hook before the intro. Catches the patterns that make AI content feel like AI content.

Decisions and tradeoffs

Single host over two hosts. Tested every OpenAI and ElevenLabs voice. All two-voice combinations sounded detectably AI. A single male voice with per-line emotional directions sounds more natural. The script uses rhetorical questions instead of a second host.

Never overwrite inputs. Each pipeline step outputs to a new file. We lost a 30-minute generated script early on by running normalize and merge in-place. Now enforced by the tools refusing identical input/output paths.

Frame-by-frame video over concatenation. Early versions concatenated ffmpeg segments, causing audible stuttering and timing drift. Now the video is built frame-by-frame—one continuous audio track with images placed at exact timestamps.

Verification at every layer. Research verification, script-vs-research fact-checking, number normalization tests, pronunciation fixing, voice consistency detection, visual spelling verification. The project has more verification code than generation code.

Learnings

AI doesn’t enforce its own rules. Instructions say “never guess.” It guesses anyway—wrong timestamps, wrong slide content, wrong diagram labels. Adding more instructions didn’t help. The fix: encode every rule as a test or pipeline gate. 8 automated quality checks block bad output before it ships. Instructions are suggestions. Code is enforcement.

Verification matters more than generation. The project has more verification code than generation code. Research sourcing, claim checking, voice consistency detection, visual timing verification, script anti-pattern scanning, adjacent-block contamination checks. AI-generated content without a trust layer is automated misinformation.

The pipeline is the product. The episodes are outputs, but the real value is the system: research any topic, verify claims, produce platform-optimized content automatically. One voicing pass produces Spotify audio, YouTube video, and blog posts. The marginal cost of a new episode is $2–3 TTS + ~$0.05 LLM.

What’s next

Multi-agent research. Parallel sub-agents research different angles (economic, technical, geopolitical, human impact) simultaneously, producing deeper coverage per topic. Quality metrics track claim density and source diversity across episodes.

Short-form clips. Automated vertical crop of Manim visuals + audio segments for YouTube Shorts and TikTok. Infrastructure built, needs per-visual optimization for 9:16 framing.
